# Сайт ООО «ЛСТ ПРО» для Netlify

Статический сайт-визитка ИТ-компании с разделом сведений для размещения на официальном сайте в целях исполнения требований Приказа Минцифры России от 02.06.2025 № 511.

## Структура

- `index.html` — основная страница сайта
- `style.css` — стили
- `script.js` — мобильное меню и текущий год
- `assets/` — SVG-иллюстрации и favicon
- `netlify.toml` — базовые настройки Netlify и security headers

## Публикация на Netlify

1. Создайте репозиторий GitHub.
2. Загрузите все файлы из архива в репозиторий.
3. В Netlify выберите Add new site → Import from Git.
4. Выберите репозиторий.
5. Build command оставьте пустым.
6. Publish directory: `.`
7. Нажмите Deploy site.

## Форма заявки

Форма настроена через Netlify Forms. После деплоя Netlify автоматически начнет принимать заявки из формы `request`.

## Важно

Перед публикацией рекомендуется проверить сведения с юристом или специалистом по аккредитации ИТ-компаний, а также при необходимости добавить основной ОКВЭД организации из ЕГРЮЛ.
